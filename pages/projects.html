WEB DEVELOPMENT

As the current website director for a student organization at University of Michigan, called EcoData, I along with the previous website director created a new website for the organization. We used HTML and CSS to create it, and Figma to design the layout!

I lead a project team to create a website called The Green Trade that allows users to trade clothing with people nearby. The goal is to promote sustainable fashion and offer a no-cost alternative to buying new clothes. The website is still in progeses but the basic components are functioning! You are able to post items you want to trade, and view other items that are available. This website is a React application set up using Google Firebase for the data collection, storage, and authentication, and utilizes HTML, CSS, and JavaScript for the front end, and Bootstrap.js for additional styling! We also used Figma for desiging!
